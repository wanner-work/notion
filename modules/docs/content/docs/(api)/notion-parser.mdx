## Usage

```ts
import { NotionParser, NotionQuery } from '@wanner.work/notion';

const query = new NotionQuery(process.env['NOTION_API_KEY']!);

const page = await query.retrievePageInformation('your-page-id');
const title = NotionParser.getPageTitle(page)
```
